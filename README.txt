== Description ==

Highslide4wp Mod

This is a modifyed version of http://www.neoease.com/ Highslide4wp – This version got support for gallery/slideshow function.

The only thing you got to do is to install, the addon as normal and then got to NextGen Gallery >> Options >> Effects >> “Link Code line”

And type in the following in the box

class="highslide" onclick="return hs.expand(this, galleryOptions, { slideshowGroup: %GALLERY_NAME% })"

And it shut work! :D Simple as that and you get the nice emotion function like normal Highslide4wp

[Highslide v. 4.0.5] (http://highslide.com/)

== Installation ==


1. Unzip archive to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Install NextGen Gallery
4. Go to NextGen Gallery >> Options >> Effect >> "Link Code line"
5. Insert the following in the box - class=”highslide” onclick=”return hs.expand(this, galleryOptions, { slideshowGroup: %GALLERY_NAME% })”
6. And is shut work!
